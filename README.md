# OOP Console Projects - C# & Python

A collection of real-world console applications developed in **C#** and
**Python** to strengthen Object-Oriented Programming (OOP), file
handling, data structures, algorithms, exception handling, and
problem-solving skills.

## Projects Included

1.  **Library Management System** - Book management, borrowing, returns,
    file persistence, and custom sorting.

2.  **Bank Account & Transaction Ledger** - Banking operations,
    transaction history, polymorphism, exception handling.

3.  **Employee & Payroll Manager** - Employee records, payroll
    calculation, CSV parsing, payslip generation.

4.  **Inventory & Order Processing System** - Inventory tracking, order
    processing, custom Stack & Queue.

5.  **Student Result Processing System** - Student grading, report
    cards, ranking, validation.

6.  **Log File Analyzer** - Log parsing, error detection, alerts,
    statistics.

7.  **Task & Ticket Tracker** - Ticket management, priority sorting,
    Kanban-style grouping.

8.  **Contact Book Management System** - Contact management, validation,
    searching, duplicate detection.
